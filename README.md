# Dr Heba El-Fiqi, academic website

Static multi-page site for GitHub Pages. Eight pages: Home, Research, Publications,
Software, People, Teaching, Service, Contact, with shared assets in /assets.

## Publish
1. Create the repository hebaelfiqi.github.io (Public).
2. Upload every file and the assets folder to the repository root (index.html at top level).
3. Settings > Pages: Deploy from branch, main, / (root).
4. Live at https://hebaelfiqi.github.io after a minute or two.

## Updating
Edit any file on GitHub with the pencil icon and commit; the site rebuilds automatically.
