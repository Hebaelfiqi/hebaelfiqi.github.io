Open-source libraries and infrastructure

Software is a first-class output here: it makes the research reproducible, gives others a benchmark, and provides the testbed that validates distributed methods.

Shepherding Library

A modular, open-source framework for modelling context-aware shepherding behaviours in multi-agent systems, with a set of agent behaviours and a simulation environment for evaluating heterogeneous agent interactions.

Swarm Simulation Testbed

A reproducible evaluation environment with standard scenarios and metrics for distributed guidance methods, used to validate collective behaviour rigorously in simulation.

Open science

Reproducibility as a default

Releasing the infrastructure behind the papers is a deliberate part of how I work: it lets others verify, reuse, and extend the methods, reduces onboarding time for new researchers and students, and turns each publication into something the community can build on rather than only read.
